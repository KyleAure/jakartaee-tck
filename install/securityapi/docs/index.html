Jakarta EE Security TCK 2.0 Documentation

The Jakarta EE Security TCK 2.0 documentation bundle includes the following TCK documentation:

  • The Jakarta EE Security TCK 2.0 Release Notes provide basic information about the test harness, the test suite and the platforms tested. This document is also included with the TCK itself.

  • The Jakarta EE Security Technology Compatibility Kit 2.0 User's Guide (HTML, PDF) describes how to install, set up, and run the Jakarta EE Security TCK 2.0. It also describes certification and compatibility requirements for the Jakarta EE Security TCK 2.0.

  • Javadoc Assertions lists all the javadoc assertions that are tested by the Jakarta EE Security TCK.

  • Specification Assertions lists all the specification assertions that are tested by the Jakarta EE Security TCK.
