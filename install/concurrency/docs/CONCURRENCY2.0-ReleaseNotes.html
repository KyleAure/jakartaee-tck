Jakarta EE Concurrency Technology Compatibility Kit, Version 2.0

Release Notes, May 2021

Kit Contents

The Jakarta EE Concurrency, Version 2.0 Technology Compatibility Kit (TCK) includes the following items:

  • Jakarta EE Concurrency TCK tests signature, API, and Spec tests:

    • A signature test that checks that all of the public APIs are supported in the Jakarta Concurrency Version 2.0 implementation that is being tested

    • API tests for all of the public APIs under the jakarta.enterprise.concurrent package

    • Spec tests for test assertions in the Jakarta EE  Concurrency Utilities for Jakarta EE 2.0 Specification

Jakarta EE Concurrency TCK Facts

The test suite bundle contains the following:

  • 71 total tests
  • Signature Test

Platform Notes

The Jakarta EE Concurrency TCK tests have been built with JDK 8 (1.8) and tested with JDK 8 (1.8) and 11.

The Jakarta EE Concurrency TCK tests have been run on the following platforms:

  • CentOS Linux 7, Alpine Linux v3.12

The Jakarta EE Concurrency TCK tests have been run against a Concurrency 2.0 Compatible Implementation.
